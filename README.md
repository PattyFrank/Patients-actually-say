# What Patients Actually Say — 2026 Sentiment Report

Interactive single-page landing site for PatientPartner's 2026 lead magnet.

## Structure
- `index.html` — main page, 12 sections
- `styles.css` — responsive styles + animations
- `script.js` — scroll animations, counters, interactive charts/toggles
- `assets/` — PatientPartner logo files

## Deploy
Drop the whole folder into any static host (Vercel, Netlify, GitHub Pages, S3).
No build step required.
